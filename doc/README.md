# GO Commands

```bash
go build
go mod init example.com/command-line
go get github.com/urfave/cli
go test
go test ./...
go test -v ./...
go test --cover
go test --cover ./...
go test --coverprofile cobertura.txt
go tool cover --func=cobertura.txt
go tool cover --html=cobertura.txt
go mod init example.com/fundamentos-adicionais
```
